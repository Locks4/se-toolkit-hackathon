# 🚀 VM Quick Start

## Only 2 Commands Needed!

### Step 1: Setup (Run once)
```bash
chmod +x setup.sh && ./setup.sh
```

### Step 2: Start App
```bash
chmod +x start.sh && ./start.sh
```

**That's it!** The app will start and show you the URL to access it.

---

## To Stop the App
```bash
./stop.sh
```

---

## What This Does

1. **setup.sh** - Installs Python, Node.js, and all dependencies
2. **start.sh** - Starts both backend and frontend servers
3. **stop.sh** - Stops all servers

The app will be accessible at: `http://YOUR_VM_IP:80`
