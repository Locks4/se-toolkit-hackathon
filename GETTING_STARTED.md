# 🎯 Getting Started - 3 Simple Steps

## Step 1: Double-click `setup.bat`
This will **automatically** install everything you need:
- ✅ Python (if missing)
- ✅ Node.js (if missing)
- ✅ All project dependencies

**That's it!** The setup script handles everything.

## Step 2: Double-click `start.bat`
This starts both servers for you:
- Backend (API server on port 8000)
- Frontend (Web app on port 3000)

Wait a few seconds, and your browser will open automatically!

## Step 3: Register and Start Using
1. Create an account (email + password)
2. Add your first goals
3. Complete them daily to earn gems
4. Build your streak!

---

## That's All You Need to Do!

No manual Python installation required. The setup.bat does it automatically.

### Troubleshooting

**If setup.bat shows an error:**
1. Make sure you have internet connection (it downloads Python/Node.js if needed)
2. Try running it again
3. If it still fails, follow the manual installation steps in INSTALL.md

**If start.bat shows errors:**
1. Make sure you ran setup.bat first
2. Close all terminals and try again
3. Check that both servers started successfully
