# ✅ Goal Creation - How It Works Now

## 🎯 Default Behavior: One-Time Goals

**By default**, when you add a goal, it's for **ONE SPECIFIC DAY** only.

The date is **automatically determined by which day you're viewing** on the calendar!

### Example:
1. Click on **April 15th** on the calendar
2. Click **"+ Add Goal"**
3. Enter "Dentist Appointment"
4. Leave the checkbox **unchecked** (default)
5. Click "Add One-Time Goal"
6. ✅ Goal appears **only on April 15th**

---

## 📅 Optional: Make It Daily

If you want a goal to repeat **every day**, check the box!

### Example:
1. Click on **any day** on the calendar
2. Click **"+ Add Goal"**
3. Enter "Exercise"
4. ✅ **Check the box** "📅 Make this a daily recurring goal"
5. Click "Add Daily Goal"
6. ✅ Goal appears **every day** until you delete it

---

### When Checkbox is CHECKED:
- Hint text disappears
- Button says: **"Add Daily Goal"**
- Goal appears every day

---

## 🗓️ How Date Is Determined

The date comes from **which day you're currently viewing**:

### Scenario 1: Viewing Today
- You haven't clicked any calendar day
- OR you clicked the "📍 Today" button
- **Result:** Goal is added for **today's date**

### Scenario 2: Viewing a Calendar Day
- You clicked on **April 20th** on the calendar
- Day view opens showing April 20th
- **Result:** Goal is added for **April 20th**

### Scenario 3: Navigating Days
- You used **← → arrows** to go to April 25th
- **Result:** Goal is added for **April 25th**

## 💡 Use Cases

### One-Time Goals (Default, checkbox UNCHECKED):
- "Dentist appointment" - April 15th only
- "Submit project" - April 20th only
- "Birthday party prep" - May 1st only
- "Team meeting" - April 18th only

### Daily Goals (Checkbox CHECKED):
- "Exercise" - Every day
- "Read 30 minutes" - Every day
- "Meditate" - Every day
- "Drink 8 glasses of water" - Every day

---

## 🎨 Visual Indicators

In the goals list:
- **No badge** = One-time goal (for a specific day)
- **Date shown** when viewing that specific day

In day view:
- All goals for that day are shown
- You can complete them for that specific date
- Subgoals work for both one-time and daily goals

|----------|-------------|-----------|---------|

## ✨ Key Points

1. **Default is one-time** - No checkbox needed for single-day goals
2. **Date from calendar** - Automatically uses the day you're viewing
3. **Check for daily** - Only check the box if you want it to repeat
4. **Flexible** - Mix one-time and daily goals as needed
5. **Subgoals work for both** - Add subgoals to any goal type
